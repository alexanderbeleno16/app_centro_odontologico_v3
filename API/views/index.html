Rutas de la APPI

/ getAll / < tablaName>

Esta ruta retorna los datos de una tabla en especifico, usted podrá elegir la tabla que desee ver, a continuación se muestran las posibles acciones que usted podrá hacer.

  • http://127.0.0.1:3000/getAll/paciente/
  • http://127.0.0.1:3000/getAll/odontologo
  • http://127.0.0.1:3000/getAll/tratamiento
  • http://127.0.0.1:3000/getAll/consulta
  • http://127.0.0.1:3000/getAll/factura

/ getAllByPar / < tablaName> / < columna> / < valor>

Esta ruta retorna registros de una tabla en especifico, usted podrá elegir la tabla, la colimna y el valor que desee ver.
Usted debe especificar los campos "tabla", "columna" y "valor".

  • http://127.0.0.1:3000/getAllByPar/

/ add_contact / < tablaName>

Con esta ruta usted podrá insertar registros a la base de datos, de acuerdo a la tabla que quiera, para ello requiere de un .json, recomendamos usar el software insmondia, para la inserción de los datos.

/ update_contact / < tablaName> / < valor id>

Con esta ruta usted podrá actualizar registros ya existentes en la base de datos, de acuerdo a la tabla que quiera, para ello requiere de un .json, recomendamos usar el software insmondia, para la actualización de los datos.

/ delate_contact / < tablaName> / < valor id>

Para eliminar solo necesita conocer el ID campo único de cualquiera de las tablas a la cual desee eliminar un registro, a continuación se muestra un ejemplo de cómo se utiliza la ruta para un paciente con ID, 2.

  • http://127.0.0.1:3000/dalate_contact/paciente/2
